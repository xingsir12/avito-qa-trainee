## Требования

- Java 17

## Быстрый старт

```bash
# Клонировать репозиторий
git clone <your-repo-url>
cd avito-qa-trainee/Task2

# Запустить тесты
mvn clean test

# Сгенерировать Allure отчет
mvn allure:report

# Открыть Allure отчет в браузере
mvn allure:serve
